# META TAGS TO ADD TO index.html

## After you get verification codes from Google & Bing:

### Google Search Console Verification:
Find this line in index.html `<head>` section:
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

Add Google verification meta tag AFTER it:
```html
<meta name="google-site-verification" content="YOUR_GOOGLE_CODE_HERE">
```

### Bing Webmaster Tools Verification:
Add Bing verification meta tag:
```html
<meta name="msvalidate.01" content="YOUR_BING_CODE_HERE">
```

### Example (after adding both):
```html
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="google-site-verification" content="abc123def456...">
<meta name="msvalidate.01" content="xyz789ghi012...">
<meta name="description" content="...">
```
